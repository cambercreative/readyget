# Ready Get

Marketing site for [readyget.app](https://readyget.app)

## About

Ready Get is the company behind:
- **[Postie](https://trypostie.com)** - Send real postcards from your phone
- **Fable** - Your stories, beautifully told (Coming Soon)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
```

## Deployment

Deployed to Vercel at readyget.app

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS v4
