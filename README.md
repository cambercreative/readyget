# Ready Get

Marketing site for [readyget.app](https://readyget.app)

## About

Ready Get builds apps that help you connect and preserve what matters most:

- **[Postie](https://trypostie.com)** - Send real postcards from your phone
- **[Fable](https://tryfable.com)** - Capture your life story with AI-powered voice interviews

## Development

```bash
npm install
npm run dev
```

## Build

Static export for deployment:

```bash
npm run build
```

Output is generated in the `out/` directory.

## Deployment

Deployed to Vercel at readyget.app
